# React Node InSim - minimal example

## Requirements

- [Node.js](https://nodejs.org/) 14 or higher

## Installation

```shell
npm install
```

## Development build

```shell
npm run dev
```

The app connects to `127.0.0.1:29999` by default.

## Production build

```shell
npm run build
npm run serve:production
```
